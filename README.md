### About Me

Computer Science student at FPT University, currently interning at **FPT Software**. Core strength is **Java Spring Boot** — designing RESTful APIs, securing services with JWT, managing relational and document databases, and building production-ready backend systems with clean architecture. Equally comfortable across the full stack with React, and experienced with low-code delivery (OutSystems) from enterprise client work.

Outside of code, I founded **The Dreamers** — a student-run charity organization — and have led development teams of up to 5 people across multiple real-world projects, plus hosted university events for 600–1000+ attendees.

### Currently

- 🔭 Building **The MC Hub** — a two-sided marketplace connecting event MCs with clients (Spring Boot 3.3, Java 21 Virtual Threads, MongoDB Atlas, WebSocket/STOMP, React Native)
- 🎓 Just shipped **ThreadLearn** — fine-tuned Qwen2.5-Coder-1.5B + BM25 RAG pipeline that auto-fixes JavaScript concurrency bugs, reaching 75% pass rate vs. 30% for zero-shot GPT-3.5-turbo
- 📚 Going deeper into microservices architecture and distributed systems
